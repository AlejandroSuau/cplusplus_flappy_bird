* Build and execute (For Windows):
[Hint] SDL2.dll should be on executable folder.

>> cmake -B build
>> cd build
>> cmake --build .
>> .\Debug\FlappyBird.exe

>> rm -r build

* Build and execute (MACOSX):
>> cmake -B build
>> cd build
>> make
>> ./FlappyBird
